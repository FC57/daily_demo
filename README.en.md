- Execute typescript files and specifications of submitting

> üåê The browser does not support typescript files and will only treat them as files to trigger download behavior. The typescript package needs to be installed and compiled into a JS file using its built-in tsc command-line tool.
>
> üí° Due to VSCode's support and highlighting of typescript syntax, files can be viewed directly in VSCode.
>
> üíª Execute the ts file using the ts-node and typescript libraries, configure the tsconfig. json file, and execute it as the `npx ts-node <file path>`. If ts-node is installed globally, npx can be omitted.

üß© In this case, the following terminal command can be run to execute the ts file.

```bash
# 'typescript/index.ts' can be replaced with the executable file path

# pnpm
pnpm ts typescript/index.ts

# npm
npm run ts typescript/index.ts
```
